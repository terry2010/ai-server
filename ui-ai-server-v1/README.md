# AI Server UI 演示

这是一个为本地AI服务管理程序设计的UI界面演示，用于管理Dify、n8n、RAGFlow和OneAPI等AI服务。

## 功能特性

- 响应式设计，支持Mac和Windows跨平台
- 现代科技风格界面，简洁美观
- 模块化管理界面
- 实时状态显示
- 系统设置配置

## 页面结构

1. **首页**
   - 通栏广告图
   - 各模块状态卡片（启动/停止状态、端口信息）
   - 快捷操作按钮

2. **模块设置页面**
   - Dify设置
   - n8n设置
   - RAGFlow设置
   - OneAPI设置

3. **系统设置页面**
   - 随系统启动配置
   - 自动启动模块配置
   - 界面风格选择

## 技术实现

- 纯HTML/CSS/JavaScript实现
- 无外部依赖
- 响应式设计
- 现代CSS特性（Flexbox, Grid）

## 使用方法

直接在浏览器中打开 `index.html` 文件即可查看演示效果。
